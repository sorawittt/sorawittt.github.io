CSS Specificity คือ

ค่าบางอย่างที่จะจะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ

การคำนวณค่า CSS Specificity

  • Inline-style มีค่าความสำคัญ 1,0,0,0 ซึ่งมีค่าความสำคัญสูงสุด
  • ID ค่าความสำคัญคือ 0,1,0,0
  • Class ค่าความสำคัญคือ 0,0,1,0
  • Element ต่าง ๆ เช่น h1, h2, div หรือ p ค่าความสำคัญคือ 0,0,0,1

การรวมค่านั้นจะดูจากจำนวนแต่ละแบบมารวมกัน ซึ่ง 0,1,0,0 นั้นก็มีค่ามากกว่า 0,1,12,0 เหมือนต่อให้มีกี่ class ก็แล้วแต่ แต่ถ้ากำหนด ID มาให้ตัวนึงก็จะยึด style จาก ID ในทันที

ตัวอย่างการรวมค่า

  • ul#nav li.active a มีค่าเท่ากับ 0,1,1,3
    ID (0,1,0,0) + class (0,0,1,0) + element (0,0,0,3)

  • <li style="color:#B23D61"> มีค่าเท่ากับ 1,0,0,0
    Inline-style (1,0,0,0)

ตัวอย่างการใช้งาน

Html

                <div id="page">
                    <div class="box">
                      <p>Hello World</p>
                    </div>
                </div>

Css

                #page .box {
                  color: green;
                }
                #page div {
                  color: red;
                }
